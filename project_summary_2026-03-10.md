## 1. 프로젝트 목적
배전계통에서 OLTC와 ESS의 협조제어를 검토하기 위한 시뮬레이터를 구축하고, 부하 및 재생에너지 변화에 따라 전압과 선로용량이 어떻게 바뀌는지 분석하는 것이 목표였습니다.

## 2. 지금까지 한 일 요약

### 2.1 구조 개편
기존 단일 파일 구조를 다음과 같이 역할별로 분리했습니다.
- `DL.py`: 사용자 인터페이스
- `sim_engine.py`: 공통 엔진
- `coordinated_engine.py`: 고도화 엔진
- `limit_finder.py`: 배치 실행 스크립트

### 2.2 시뮬레이션 기능 고도화
- 단일 시나리오 실행 유지
- 자동 민감도 분석 기능 추가
- 세 가지 시나리오 지원
  - 부하 증가
  - 재생에너지 증가
  - 부하구간별 재생에너지 증가
- 시간 간격을 1분 또는 10분 등으로 조정 가능하게 변경

### 2.3 제어 알고리즘 개선
- 전압 제약과 선로용량 제약을 동시 고려
- OLTC 1대 + ESS 1대 구조로 명확화
- ESS 효율 반영
- 협조제어 파라미터를 UI에서 수정 가능하도록 구성

### 2.4 UI 개선
- 설정 영역 구조화
- 알고리즘 수정 탭 추가
- 실행 중 입력 잠금
- 중단 후 복구용 재실행/초기화 도입
- 결과 라벨링 강화

### 2.5 결과물/보고서 개선
- Excel 결과 다운로드
- Word 보고서 생성
- 보고서에 그래프, 회차별 결과, 운영 범위, 진단 내용 반영

## 3. 사용자 요청과 연결된 반영 사항
- 부하를 점진적으로 올리는 자동화 시뮬레이션 반영
- 재생에너지 증가 시나리오 반영
- 부하구간별 재생에너지 증가 시나리오 반영
- 보고서에서 결과 원인 분석 포함
- 시뮬레이션 중 설정 변경 방지 반영
- 화면 중단 후 복구 흐름 정리

## 4. 현재 생성 가능한 주요 산출물
